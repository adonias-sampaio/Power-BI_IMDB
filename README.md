# Power-BI_IMDB

In this dashboard, I sourced the data from Kaggle, which provides a list of the top 250 ranked films on IMDb.

The IMDb Top 250 is one of the most famous and highly-regarded movie rankings globally, created and maintained by the Internet Movie Database (IMDb). It's a list that represents the top-rated 250 feature films based on user ratings.
